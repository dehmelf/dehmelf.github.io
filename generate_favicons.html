Favicon Generator for |F⟩⟨D|

This page generates PNG favicons from the SVG. Right-click each canvas and "Save image as..." to save the favicon files.

Save them as:

  • 16x16 → favicon-16x16.png
  • 32x32 → favicon-32x32.png
  • 180x180 → apple-touch-icon.png

After downloading:

  1. Save the downloaded files to static/ directory (replace existing files)
  2. Clear your browser cache (Ctrl+Shift+Delete or Cmd+Shift+Delete)
  3. Hard refresh the page (Ctrl+F5 or Cmd+Shift+R)
  4. Or try opening the site in an incognito/private window
